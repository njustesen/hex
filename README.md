# hex

Hex game engine package.

## Installation

Install in development mode:

```bash
pip install -e .
```

Or install normally:

```bash
pip install .
```

## Usage

After installation, you can import from the hex package:

```python
from hex import GameEngine, EngineConfig, colors
from hex.map import HexGridMap, TileGridMap
from hex.viewport import Viewport
# etc.
```
